# Rock Paper Scissors - Java
A Rock Paper Scissors game developed in Java using Swing.

## Features
- Graphical User Interface (Java Swing)
- Three difficulty levels (Easy, Medium, Hard)
- Round selection
- Score tracking
- Computer AI with different behaviors based on difficulty

## Technologies
- Java
- Java Swing
- Apache NetBeans

## How to Run
1. Clone this repository.
2. Open the project in Apache NetBeans.
3. Run the `GameGUI` class.
